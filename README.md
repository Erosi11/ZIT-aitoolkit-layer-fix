# ZIT-aitoolkit-layer-fix
This is a quick fix for missing lora key issues in comfyui from ZIT loras generated with aitoolkit. Built upon work from FOK wrote the main codes and PAseer made a simple UI which can easliy converts the Ai-toolkit_LORA into Z-image-ComfyUI-LORA. I simply translated to English and added batch processing as well as a separate output folder selection
